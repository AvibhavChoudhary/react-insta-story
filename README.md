# Instagram Story Clone

## Live Demo

[View the live demo](https://react-insta-story-hazel.vercel.app/)

## Tech Stack

- **Frontend Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: CSS
- **Deployment**: Vercel

## Features

- Instagram-like story viewer interface
- Smooth transitions between stories
- Progress bar for story duration
- Responsive design
- Touch/click navigation

## Getting Started

### Installation

1. Clone the repository

```bash
git clone (https://github.com/AvibhavChoudhary/react-insta-story)
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

## Design & Technical Choices

### Scalability Considerations

- Modular component architecture for easy feature additions
- TypeScript for better type safety and maintainability
- CSS modules to prevent style conflicts
- Responsive design patterns for various screen sizes

### Code Organization

- Component-based architecture
- Separate concerns for UI, logic, and data management
- Reusable components for story elements
- Clear file structure for better maintainability
